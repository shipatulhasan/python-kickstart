# Program Explanation

- I defined the variables according to the given requirements.
- I used `if/elif/else` statements to determine the appropriate access tier based on the user's input.
- I used a `while` loop to make the developer status input more user-friendly.
- The user must enter either `y` or `n` for the developer status.
- If any other value is entered, the program displays an error message and keeps asking for a valid input.
- The program will not proceed to the next step until a valid response (`y` or `n`) is provided.
